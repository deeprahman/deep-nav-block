A dynamic navigation menu block for WordPress Gutenberg editor.

== Description ==

Deep Nav Block adds a customizable navigation menu block to the WordPress block editor. Key features:

Unlike the native Full site Editor Nav block, this plugin offers a dropdown feature for mobile navigation.

* Dynamic menu generation from WordPress menus
* Multi-level navigation support
* Customizable styling options
* Mobile-responsive design
* Easy integration with existing themes

== Installation ==

1. Upload `deep-nav-block` folder to `/wp-content/plugins/` directory
2. Activate plugin through WordPress 'Plugins' menu
3. Add the Navigation Block to any page/post using the block editor
4. Configure menu settings in the block sidebar

== Frequently Asked Questions ==

= How do I create a multi-level menu? =

Use the WordPress Full Site Editor Navigation interface to create hierarchical menu structures. Go to Appearance > Editor > Navigation to build your menu. The block will automatically render all menu levels.

= Can I customize the menu styles? =

Yes, the block includes style options for colors, spacing, and typography in the block settings sidebar.

= Is it mobile responsive? =

Yes, the navigation menu automatically adapts to different screen sizes.
